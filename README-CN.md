# Spring AI Lab 

A practical, end-to-end guide to learning and building AI applications using Spring AI — from core concepts to real-world agent systems.

---

## ✨ Overview

This repository is designed to help developers understand and apply Spring AI step by step:

- Learn core concepts of AI integration in backend systems
- Build real AI-powered features with Spring Boot
- Understand prompt engineering and LLM interaction
- Create tool-using AI agents
- Explore real-world use cases

---

## 🎯 Goals

- Bridge the gap between AI concepts and backend engineering
- Provide hands-on, runnable examples
- Help backend developers (especially Java engineers) enter the AI space
- Demonstrate production-ready patterns

## 🚀 Getting Started

### 1. Prerequisites

- Java 17+
- Spring Boot 3+
- Maven or Gradle
- API Key from your LLM provider (e.g., OpenAI)

---

### 2. Setup

bash git clone https://github.com/<your-username>/spring-ai-lab.git cd spring-ai-lab

---

### 3. Configure API Key

bash export OPENAI_API_KEY=your_api_key

---

### 4. Run Example

bash cd 01-basics mvn spring-boot:run
